# Weissinger Lifting-Line Theory

This directory contains the iterative Weissinger method implementation for 3D finite-wing analysis with XFLR5 validation and aircraft comparison studies.

## Contents

- `Original Weissinger/Iterative Weissinger Script/` - MATLAB implementation of Weissinger method
- `Original Weissinger/XFLR5 Files/` - Validation data and aircraft configurations
- Result plots comparing MATLAB and XFLR5 predictions
