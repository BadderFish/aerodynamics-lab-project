# Panel Method Airfoil Analysis

This directory contains the Hess-Smith panel method implementation for 2D airfoil analysis with XFOIL validation and comparative studies of NACA airfoils and the AG26 Bubble Dancer.

## Contents

- `starting_point_HessSmith/` - MATLAB implementation of Hess-Smith panel method
- `example_script_bash/` - Python plotting scripts and XFOIL batch processing
- `info_HessSmith.pdf`, `info_xfoil.pdf`, `paper_xfoil.pdf` - Reference documentation
